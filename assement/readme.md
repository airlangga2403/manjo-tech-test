# Payment Gateway Technical Test

## Overview

Implementation of Payment Gateway API for PT Manjo Teknologi Indonesia Technical Test.

This project simulates:

1. Generate QR Transaction
2. Query Payment Status
3. Payment Notification Callback

Technology Stack:

* Java 21
* Spring Boot 3
* PostgreSQL
* Spring Data JPA+
* Lombok
* Swagger OpenAPI


PostgreSQL docker container :: docker compose up -d
